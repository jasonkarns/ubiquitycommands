ubiquitycommands/radbox by jasonkarns

A Ubiquity command for adding videos to Radbox

Requirements

  1. Firefox
  2. The Ubiquity extension
  3. A Radbox account

Installation

If you have Ubiquity installed and enabled, you should be given the option to install the Radbox command via drop down at the top of this page.

Get the Source

You can clone the project with Git by running: 

$ git clone git://github.com/jasonkarns/ubiquitycommands.git
find all my Ubiquity commands on GitHub : jasonkarns/ubiquitycommands
